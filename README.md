# LWC Button

An OSS version of lightning button with no dependencies

## Consuming

Tries to stick to standards (`onclick`) or Salesforce's API (`variant`)

```html
<lwc-button
    variant="brand"
    label="Download"
    title="Begins the download"
    onclick={handleClick}>
</lwc-button>
```
